
# GreenCamp

A website for exploring Campgrounds all around the world. 

Live project link : https://greencamp-l5y2.onrender.com/

## Run Locally

Clone the project

```bash
gh repo clone lite-4846/GreenCamp
```

Go to the project directory

```bash
cd GreenCamp
```

Install dependencies

```bash
npm install
```

Start the server

```bash
node app.js
```
